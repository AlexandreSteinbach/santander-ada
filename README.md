# Curso Digital: git

## Iniciando o Git
* comando git init

## Puxando informações remotas
* comando git clone

## Salvando alterações no Git
* comando git add .(\local)
* comando git commit -m "mensagem"
* comando git push

## Puxar informações do repositório remoto
* comando git pull

## Para fazer pull com cuidado, fazer antes:
*git fetch
*git diff

## Outros comandos
*git remote
*git diff
*git restore
